# 🌐 Back-end-Development-Capstone

## 🚀 Overview

This is a **full-stack Django-based capstone project** integrating multiple microservices:

* 📸 Pictures Service
* 🎵 Songs Service
* 🌐 Django Frontend

The application demonstrates **cloud deployment, containerization, and microservices integration**.

---

## 🛠️ Tech Stack

* Python 🐍
* Django 🌐
* Flask ⚡
* Docker 🐳
* Kubernetes ☸️
* IBM Cloud Code Engine ☁️

---

## 📂 Features

* ✅ Songs listing and modal view
* ✅ Photos gallery
* ✅ Concerts page
* ✅ Django Admin panel
* ✅ Integrated backend services

---

## ▶️ Setup Instructions

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run migrations

```bash
python manage.py migrate
```

### Start server

```bash
python manage.py runserver
```
